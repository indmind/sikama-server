Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://mage8-server.test

Authenticating requests

Authenticate requests to this API's endpoints by sending an Authorization header with the value "Bearer {token}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve the token by calling Google SignIn endpoint with debug_token as access_token value.

Authentication

APIs for managing authentication

Google SignIn

Login user and create token using Google access token.

Example request:
curl --request POST \
    "http://mage8-server.test/api/v1/auth/google-sign-in" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"access_token\": \"debug_token\",
    \"device_name\": \"Scribe\"
}"

const url = new URL(
    "http://mage8-server.test/api/v1/auth/google-sign-in"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "access_token": "debug_token",
    "device_name": "Scribe"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

User Management

Update user position

requires authentication

This will update the current user position, set to null to remove.

Example request:
curl --request POST \
    "http://mage8-server.test/api/v1/user/position/update" \
    --header "Authorization: Bearer {token}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"latitude\": -6.2145,
    \"longitude\": 106.8451
}"

const url = new URL(
    "http://mage8-server.test/api/v1/user/position/update"
);

const headers = {
    "Authorization": "Bearer {token}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "latitude": -6.2145,
    "longitude": 106.8451
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "message": "User position updated"
}

Example response (200, called with empty latitude and longitude):


{
    "message": "User position cleared"
}
